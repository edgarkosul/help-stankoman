# Товары

Раздел `/admin/products` показывает общую таблицу товаров каталога.

## Что видно в таблице

В таблице отображается список товаров с основными колонками по карточке товара. Состав колонок можно менять: часть столбцов можно скрывать, чтобы оставить только нужные поля для текущей задачи.

## Фильтры

В разделе уже есть набор фильтров для быстрого поиска нужных товаров. Если текущих фильтров недостаточно, можно заказать у разработчика добавление любых дополнительных фильтров под ваши рабочие сценарии.

## Массовые действия

Для выбранных товаров доступны массовые операции:

1. Массовое удаление.
2. Инструмент массового изменения товаров.

Эти действия позволяют быстрее вносить изменения сразу в несколько карточек без поштучного редактирования.
